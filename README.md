# bake
Compiler for CSE423.

## How to Compile
- Run cmake_script.sh in terminal.
- Go into build/src/
- Run ./bake [file.cl] > [output file]

## How to use COOL Reference Interpreter
- Make sure you have executable permission on the "cool" executable.
  - chmod a+x cool
- Run ./cool [file.cl]
  - Run with "--lex" for the lexer
